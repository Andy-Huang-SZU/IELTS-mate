# IELTS-mate 设计规范

> **注意：** 原有的 Pencil 设计文件 (`ielts-mate.pen`) 已被废弃并删除。

现在项目全面采用 **Flux Academy (Light Mode)** 设计风格。

## 核心设计文档

请参考以下文档进行前端开发：

1.  **视觉规范**：`docs/frontend-design-new.md`
    - 包含色板、排印、核心视觉实现（流体背景、磨砂玻璃、3D 球体）的代码思路。
2.  **页面规划**：`docs/frontend-pages.md`
    - 包含路由结构、Bento Grid 布局细节、各功能模块的 UI 构成。
3.  **参考素材**：`docs/assets/hero_section_template.jpeg`
    - 首页 Hero Section 的视觉目标参考图。

## 风格关键词

- **Light & Warm**: 暖米白背景，拒绝深色沉重感。
- **Ceramic Glass**: 陶瓷质感的厚磨砂玻璃。
- **Fluidity**: 交互式流体光斑背景。
- **Tactile**: 果冻般的微交互反馈。
